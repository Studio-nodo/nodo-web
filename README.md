# Studio Nodo — Landing Page

Landing page corporativa de **Studio Nodo**, estudio de diseño y desarrollo web para PYMEs y emprendedores en Argentina.

## Servicios

- **Landing Page** — Páginas diseñadas para convertir visitas en clientes, optimizadas para carga rápida y mobile-first.
- **E-Commerce** — Tiendas online completas con pasarelas de pago, gestión de stock y panel administrativo.
- **Web Corporativa** — Sitios profesionales con CMS para gestión autónoma, blog y formularios personalizados.
- **Branding** — Identidad visual completa: logotipo, paleta de colores, tipografías y manual de marca.
- **Redes Sociales** — Gestión integral de Instagram, Facebook y LinkedIn con estrategia de contenidos y métricas.
- **Logotipo** — Diseño profesional a medida con propuestas conceptuales y entrega en todos los formatos.

## Stack

- Next.js (App Router) + TypeScript
- Tailwind CSS + Framer Motion
- Deploy en Vercel
