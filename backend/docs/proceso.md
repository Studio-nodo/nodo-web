# Proceso de Trabajo de Studio Nodo

## Paso 1: Descubrimiento
Reunión inicial para entender los objetivos del proyecto, público objetivo y expectativas. Analizamos la competencia y definimos el alcance.

## Paso 2: Propuesta
Presentamos una propuesta detallada con estructura, funcionalidades, tecnologías a usar, plazos y presupuesto.

## Paso 3: Diseño
Creamos wireframes y prototipos visuales. El cliente aprueba el diseño antes de avanzar al desarrollo.

## Paso 4: Desarrollo
Implementamos el proyecto con nuestro stack tecnológico. Entregamos avances parciales para revisión y feedback.

## Paso 5: Lanzamiento
Deploy a producción, configuración de dominio, SSL y optimización de performance.

## Paso 6: Soporte
Acompañamos al cliente post-lanzamiento con soporte técnico y mantenimiento.

## Plazos típicos
- Landing Page: 1-2 semanas
- Web Corporativa: 3-4 semanas
- E-Commerce: 4-8 semanas
- Agente IA: 2-4 semanas
