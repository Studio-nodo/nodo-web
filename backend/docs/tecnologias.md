# Tecnologías que usa Studio Nodo

## Frontend
- **Next.js 16** — framework React con App Router y Server Components
- **React 19** — librería de interfaz de usuario
- **TypeScript** — tipado estático para mayor seguridad
- **Tailwind CSS** — estilos utility-first
- **Framer Motion** — animaciones fluidas
- **Three.js / React Three Fiber** — gráficos 3D en el navegador

## Backend
- **Python 3.12** con **FastAPI** — APIs REST de alto rendimiento
- **LangChain** — orquestación de agentes IA
- **Gemini 1.5 Flash** (Google) — modelo de lenguaje para el agente
- **Pinecone** — base de datos vectorial para búsqueda semántica

## Deploy e Infraestructura
- **Vercel** — deploy del frontend con CDN global
- **Railway** — deploy del backend Python
- **Cloudflare** — DNS, protección y caché
- **GitHub** — control de versiones y CI/CD

## Herramientas de Desarrollo
- **VS Code** — editor principal
- **Claude Code** — asistente IA para desarrollo

## Por qué estas tecnologías
Elegimos tecnologías probadas en producción que combinan velocidad de desarrollo, performance y escalabilidad. Priorizamos herramientas open source y cloud services con free tiers generosos para que los proyectos sean viables desde el día uno.
